Примером того как правильно организовать работу 
автоматической тестирующей системы в интернет может
служить сайт Уральского Университета 
acm.timus.ru
Для всех желающих проверить свои в решении задач по программированию
в соответствии с международными правилами ACM организован сайт
с удобным интерфейсом и большим количеством задач.

На сайте университета можно: 
1. Посмотреть информацию о системе
2. Послать на проверку свое решение одной из задач
3. Прочитать список задач
4. Посмотреть состояние автоматической системы
(какие задачи она в данных момент проверяет)
5. Обсудить решение и условия задач
6. Узнать информацию об олипиадах проводимых на данном сайте.
 
Вариант работы с автоматической системой acm.timus.ru
Этап 1. Чтение условия
Вы нажимаете на ссылку Список задач.
Выбираете ту задачу которая вам нравится. 
Читаете ее условие и пример ввода/вывода который помещен там же.
Примечание: Условия задач на либо на русском, либо на английском, 
либо на обоих языках сразу.
Этап 2. Создание исходного кода программы (решения):
Исходный код можно писать на языках Pascal или C.
// Решение задачи 1001 - 1001.cpp "Сложение 2-х чисел"
// Ввод: a b    a,b 0..10E9
// Вывод: a+b
#include < stdio.h >
int main()
{
  long int a,b;
  scanf("%d %d",&a,&b);
  printf("%d\n",a+b);
  return 0;
}
Этап 3. Компилируте и тестируете программу:
3.1. Компилируете программу:
path S:\bc45\bin;S:\bc45\include;S:\bc45\lib;
S:\bc45\bin\bcc32.exe -IS:\bc45\include -LS:\bc45\lib 1001.cpp
3.2. Для тестирования программы помещаете в файл i.txt числа 2 и 3
разделенные пробелом.
3.3. Затем запускаете программу на выполнение строкой:
1001.exe < i.txt > o.txt
3.4. При этом результат ее работы помещается в файл o.txt.
После этого остается лишь убедиться что o.txt содержит число 5.
Этап 4. Отсылаете решение на проверку acm.timus.ru
4.1. Регистируете свою команду Зарегистрироваться.
4.2. Переходите по ссылке Послать на проверку.
4.3. Отсылаете свое решение на проверку
Этап 5. Смотрите себя в рейтинге участников
Для этого преходите по ссылке Status (online)
и находите себя в общем списке.
